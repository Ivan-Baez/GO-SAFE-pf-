# 🔐 GoSafe

Aplicación Full Stack orientada a la gestión de usuarios, reservas y actividades, desarrollada utilizando tecnologías modernas del ecosistema JavaScript y TypeScript.

GoSafe permite administrar perfiles, publicaciones y experiencias desde una interfaz moderna e intuitiva, integrando autenticación, manejo de roles y persistencia de datos.

---

# 🚀 Características

- Registro e inicio de sesión de usuarios
- Gestión de perfiles
- Dashboard interactivo
- Sistema de roles
- CRUD de publicaciones y actividades
- Arquitectura backend modular
- APIs REST
- Validaciones y manejo de errores
- Persistencia de datos relacional
- Integración frontend y backend
